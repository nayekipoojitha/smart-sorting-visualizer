\# Smart Sorting Visualizer



Smart Sorting Visualizer is a web-based application that visually demonstrates how different sorting algorithms work.  

It helps users understand sorting behavior, performance, and time complexity through animated bar representations.



---



 Features



\- Visualizes Bubble Sort, Merge Sort, and Quick Sort

\- Animated bars represent array elements

\- Displays execution time for each algorithm

\- Shows element values on hover for clarity

\- Simple and interactive UI



---



\## Technologies Used



\- HTML

\- CSS

\- JavaScript

\- Data Structures and Algorithms



---



\## How It Works



\- Generates a random array of numbers

\- Each number is displayed as a vertical bar

\- Sorting algorithms rearrange bars step-by-step

\- Execution time is measured using JavaScript performance APIs



---



\## How to Run



1\. Download or clone the repository

2\. Open `index.html` in any web browser

3\. Click on a sorting algorithm button to visualize it



---



\## Use Case



This project is useful for:

\- Understanding sorting algorithms

\- Learning time complexity differences

\- Demonstrating DSA concepts visually
